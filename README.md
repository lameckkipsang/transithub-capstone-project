# TransitHub Solutions

**TransitHub Solutions** is a comprehensive, multi-page logistics and travel web platform designed to solve real-world logistical inefficiencies. It serves as a central hub connecting users with professional heavy-freight drivers, trusted moving companies, and peer-to-peer (P2P) vehicle rentals securely and efficiently.


##  Features

*   **P2P Car Hiring:** A dedicated marketplace for users to rent out their personal vehicles safely or browse available cars with verified star ratings and image uploads.
*   **Transit Goods Transport:** Long-distance lorry transportation booking system featuring strict compliance policies for legal goods and dynamic vehicle sizing.
*   **Movers Brokerage:** A step-by-step connection service to premium moving companies, featuring an interactive, pure-CSS instant quotation calculator for various house sizes (Bedsitter, 1 Bedroom, 2 Bedroom).
*   **M-Pesa Payment Workflow:** Integrated mobile money payment instructions and transaction code verification UI for booking confirmations.
*   **Custom Authentication Flow:** Distinct, responsive pages for user registration and secure login, routing to a personalized Member Dashboard.

##  Tech Stack

This project is built strictly using core web technologies:

*   **HTML5:** Use of HTML tags to create a  functional website.
*   **Tailwind CSS (v4 via CDN):** For better responsive styling.
*   **CSS:** Custom CSS styling using the style.css file.
